🛒 Shopping Cart - Online Course Platform

Welcome! This project is an interactive shopping cart for an online course platform. It was developed to strengthen my knowledge of modern JavaScript (ES6+), DOM manipulation, and implementing business logic without relying on external libraries or frameworks.

🚀 Features

📚 Add Courses: Dynamically add selected courses to the shopping cart.
🔄 Duplicate Management: If a course is already in the cart, its quantity is increased instead of creating a duplicate entry.
💰 Automatic Subtotal Calculation: Updates the total cost in real time as courses are added or removed.
🗑️ Cart Management: Remove individual courses or clear the entire shopping cart.
⚡ Dynamic Rendering: Updates only the necessary DOM elements to improve performance.

🛠️ Technologies Used

► HTML5 — Semantic structure of the application.
► CSS3 — Styling and layout.
► JavaScript (ES6+) — Business logic, DOM manipulation, and event handling.

🧠 JavaScript Concepts Applied

During the development of this project, I implemented the following concepts:

► Event Delegation to efficiently handle user interactions.
► DOM Manipulation using methods such as querySelector, createElement, appendChild, and innerHTML.
► Array Methods including map(), filter(), forEach(), find(), some(), and reduce().
► Arrow Functions and Object Destructuring.
► Template Literals for dynamically generating HTML content.
► Code organization through reusable functions and separation of concerns.
